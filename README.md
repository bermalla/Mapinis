# Croquis Map Editor

Editor web local para crear croquis simples de mapas sobre una grilla.

## Cómo correrlo

Abrí `index.html` directamente en el navegador.

No requiere servidor, build ni dependencias.

## Archivos

- `index.html`: estructura de interfaz.
- `styles.css`: estilos visuales.
- `app.js`: lógica de editor.
- `PROJECT_GUIDELINES.md`: lineamientos de diseño y programación.

## Funciones principales

- Grilla 100x72.
- Herramientas de bloques.
- Bloques 1x1 por click.
- Bloques rectangulares por click + drag.
- Selección y movimiento de bloques.
- Redimensionado con manijas.
- Texto multilínea dentro de bloques.
- Guardar/cargar JSON.
- Exportar PNG.
- Deshacer con botón y Ctrl+Z.
- Reordenar herramientas.
- Mover seleccionado con flechas.
