Mapinis

Herramientas

Un clic coloca un bloque 1x1. Si mantenés el clic y arrastrás, se crea un bloque rectangular. Las teclas 1 a 0 seleccionan las primeras 10 herramientas. Podés reordenarlas arrastrando una herramienta sobre otra.

Capas

Capa 0

Solo se edita la capa activa. Podés mostrar una capa adyacente como referencia tenue.

Propiedades del seleccionado

Doble click en un bloque para enfocar edición de texto. Texto, tamaño, ancho y alto se actualizan en tiempo real. Las manijas permiten redimensionar el bloque seleccionado.

Respaldo JSON

Si el navegador bloquea la descarga, usá Copiar JSON y guardá el texto manualmente en un archivo .json.

Controles

1–0: seleccionar herramienta.
Clic: colocar 1x1 o seleccionar bloque existente.
Doble click: enfocar edición de texto.
Clic + arrastrar en vacío: crear rectángulo.
Seleccionar/mover + arrastrar en vacío: desplazar mapa.
Arrastrar bloque seleccionado: moverlo.
Flechas: mover seleccionado 1 celda.
Supr/Backspace: borrar seleccionado.
Ctrl+Z: deshacer.
Esc: quitar selección.
R: rotar texto/flecha.
D: duplicar seleccionado.
